Sobre o projeto

O usuário deve ser capaz de:
=> ver os itens com layout otimizado de acordo com o tamanho da tela;
=> ver os efeitos hover em todos os elementos interativos da página;
=> navegar pelas páginas de projeto através da barra de navegação interna;
=> ver uma mensagem de erro se:
==> os campos "Nome", "Email" "Messagem" estiverem vazios. A mensagem deve ser "Este campo é necessário";
==> o e-mail não estiver formatado corretamente. A mensagem deve ser"Insira um endereço de e-mail válido";
